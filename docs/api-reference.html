SIMPLE_FILE

Main class for file and directory operations.

Creation

Feature Signature Description
make (a_path: READABLE_STRING_GENERAL) Create from string path.
make_with_path (a_path: PATH) Create from PATH object.

File Status

Feature Signature Description
exists : BOOLEAN Does file or directory exist?
is_readable : BOOLEAN Can file be read?
is_writable : BOOLEAN Can file be written?
is_executable : BOOLEAN Can file be executed?
is_directory : BOOLEAN Is this a directory?
is_file : BOOLEAN Is this a regular file?
is_symlink : BOOLEAN Is this a symbolic link?

File Metadata

Feature Signature Description
size : INTEGER_64 File size in bytes.
modified_timestamp : INTEGER Modification time (epoch seconds).
accessed_timestamp : INTEGER Access time (epoch seconds).
extension : STRING_32 File extension without dot.
base_name : STRING_32 Filename without extension.
file_name : STRING_32 Full filename with extension.
parent_path : STRING_32 Parent directory path.
path : PATH Full PATH object.
path_string : STRING_32 Full path as string.

Read Operations

Feature Signature Description
read_text : STRING_32 Read entire file as text.
read_lines : ARRAYED_LIST [STRING_32] Read file as list of lines.
read_bytes : ARRAY [NATURAL_8] Read entire file as bytes.
open_read Open file for streaming read.
read_chunk (a_size: INTEGER): STRING_32 Read chunk of specified size.
end_of_file : BOOLEAN Is at end of file?
close Close open file.

Write Operations

Feature Signature Description
write_text (a_content: STRING_32) Write text (overwrites existing).
append_text (a_content: STRING_32) Append text to file.
write_lines (a_lines: LIST [STRING_32]) Write list of lines.
append_lines (a_lines: LIST [STRING_32]) Append list of lines.
write_bytes (a_bytes: ARRAY [NATURAL_8]) Write binary data.
create_empty Create empty file (touch).

File Operations

Feature Signature Description
copy_to (a_destination: STRING_32) Copy file to destination.
move_to (a_destination: STRING_32) Move file to destination.
rename_to (a_new_name: STRING_32) Rename file (same directory).
delete Delete file.

Directory Operations

Feature Signature Description
list_directory : ARRAYED_LIST [STRING_32] List directory contents.
create_directory Create directory.
create_directory_recursive Create nested directories.
delete_directory Delete empty directory.
delete_directory_recursive Delete directory and contents.

Error Handling

Feature Signature Description
has_error : BOOLEAN Did last operation fail?
last_error : STRING_32 Error message from last operation.
clear_error Clear error state.

SIMPLE_FILES

Batch file operations and pattern matching.

Creation

Feature Signature Description
make (a_directory: STRING_32) Create for directory.

Find Operations

Feature Signature Description
find_by_extension (a_ext: STRING): ARRAYED_LIST [SIMPLE_FILE] Find files by extension.
find_by_pattern (a_pattern: STRING): ARRAYED_LIST [SIMPLE_FILE] Find files by glob pattern.
find_recursive (a_pattern: STRING): ARRAYED_LIST [SIMPLE_FILE] Find files recursively.
all_files : ARRAYED_LIST [SIMPLE_FILE] Get all files in directory.
all_directories : ARRAYED_LIST [SIMPLE_FILE] Get all subdirectories.

SIMPLE_PATH

Path manipulation utilities.

Creation

Feature Signature Description
make (a_path: STRING_32) Create from string.
make_from_path (a_path: PATH) Create from PATH.

Path Operations

Feature Signature Description
joined (a_component: STRING_32): SIMPLE_PATH Join path component.
parent : SIMPLE_PATH Parent directory path.
name : STRING_32 Full path as string.
file_name : STRING_32 Just the filename.
extension : STRING_32 File extension.
is_absolute : BOOLEAN Is path absolute?
is_relative : BOOLEAN Is path relative?
normalized : SIMPLE_PATH Normalized path (resolve . and ..).

Part of the simple_* ecosystem of focused, single-purpose Eiffel libraries.
